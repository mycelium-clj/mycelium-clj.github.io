# How you can help

Mycelium is an open source project and there are many ways to get involved.

## Report Bugs

The project is [hosted on GitHub](https://github.com/mycelium-clj/mycelium). If you have feature requests or would like to submit improvements, don't hesitate to open an issue or submit a pull request.

## Help Improve Documentation

The source for this site along with the associated documentation is hosted on [GitHub](https://github.com/mycelium-clj/mycelium-clj.github.io).

If you wish to update an existing documentation page, simply edit the Markdown
file for the page [here](https://github.com/mycelium-clj/mycelium-clj.github.io/tree/main/resources/md).

To add a new topic simply create a new Markdown file under [resources/md](https://github.com/mycelium-clj/mycelium-clj.github.io/tree/main/resources/md)
directory, and update the list of topics [here](https://github.com/mycelium-clj/mycelium-clj.github.io/blob/main/resources/docpages.edn).
